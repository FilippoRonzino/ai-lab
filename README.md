# AI Lab 2024 Repository

This repository contains materials for the AI Lab 2024, held at Bocconi University. The project authors are: Ghirardo Edoardo, Iannone Giuseppe, Hoche Francois, Ronzino Filippo A., Tofanelli Elisa

## Introduction

This repository hosts all the necessary files and documentation pertaining to the AI Lab project. While the data files are strictly confidential and not public, the project structure, methodologies, and results are available in the accompanying Jupyter notebook.
